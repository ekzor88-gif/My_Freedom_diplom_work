# СмартСнаб (Smart Supplier SaaS)

Современная система управления складскими запасами с интегрированной ИИ-аналитикой, построенная на базе React и интегрированная с оркестратором n8n.

## 🚀 Особенности

- **Управление складом**: Мониторинг запасов, критических порогов и продаж.
- **AI-Аналитика**: Автоматический анализ состояния склада, выявление дефицитов и лидеров продаж с помощью искусственного интеллекта.
- **Управление заказами**: Генерация автоматических email-заявок для поставщиков.
- **Интеграция с n8n**: Бесшовная связь бэкенда и фронтенда через вебхуки n8n.
- **Современный UI**: Отзывчивый интерфейс с поддержкой светлой и темной тем, плавными анимациями (Framer Motion) и графиками (Recharts).

## 🛠 Технологический стек

- **Фреймворк**: React 18 + Vite
- **Роутинг**: React Router v6
- **Стилизация**: Tailwind CSS
- **Взаимодействие с API**: Axios
- **Визуализация данных**: Recharts
- **Анимации**: Framer Motion
- **Иконки**: Lucide React

## 📦 Установка и запуск локально

1. Склонируйте репозиторий:
   ```bash
   git clone <url_вашего_репозитория>
   cd smart-supplier-saas
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Настройте переменные окружения:
   Скопируйте файл `.env.example` в `.env` и укажите ваши URL вебхуков n8n:
   ```bash
   cp .env.example .env
   ```

4. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

5. Откройте в браузере `http://localhost:5173`.

## 🏗 Сборка для продакшена

Для сборки оптимизированной версии приложения выполните:
```bash
npm run build
```
Готовые статические файлы будут находиться в папке `dist/`.
